## Track 3: Duplicate Root Cause

Built `vcm_os/cli/diagnose.py`:
- Shows total memories, by-type breakdown
- Shows duplicate analysis
- Shows per-event memory count
- Shows canonicalization skips

**Root cause found:** `hash(raw_text) % 10000` in `_evt()` is non-deterministic across Python runs (PYTHONHASHSEED). Events re-ingested with different IDs, bypassing `ON CONFLICT`. `data/vcm_os.db` accumulated garbage.

**Fix:** Replaced with `hashlib.md5(raw_text.encode()).hexdigest()[:8]`.

After fix + DB cleanup:
```
Total memories: 10
Duplicate instances: 0 (cross-type by design)
Canonicalization skips: 0
```
